**Как запустить расширение в тестовом режиме:**

1. Скачать данный репозиторий локально
2. Перейти по ссылке browser://extensions/
3. В верхнем правом углу включить "Режим разработчика"
4. Нажать "Загрузить распакованное расширение"
5. Выбрать папку, содержащую файлы из репоизтория
